## INSTALLATION

**Requirements:**
- pipenv install fabric3 
- install docker
- install docker-compose

**Commands:**
1. git pull

2. fab build_dev

3. edit .env file

4. fab init_dev

Will be asked to create superuser

5. fab up_dev

6. fab down

To delete containers

7. fab test 

To run Unittests
